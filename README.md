# ✈️ Flight Fare Prediction System

A Machine Learning web app that predicts flight ticket prices based on user inputs.

## 🚀 Live Demo
https://flight-fare-prediction-ibd26fd5yod9bfd3ha9dtx.streamlit.app

## 📌 Features
- Predict flight ticket prices
- User-friendly UI using Streamlit
- Real-time prediction

## 🛠 Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas, NumPy

## 📊 Model
- Random Forest Regressor
- Accuracy: ~83%

## 📂 Project Structure
- app.py → Streamlit UI
- main.py → Model training
- model_features.pkl → Feature columns
